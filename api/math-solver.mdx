## Overview

The Math Solver API uses Claude AI to solve mathematical problems with detailed step-by-step explanations. It supports various problem types including algebra, calculus, geometry, trigonometry, and statistics.

## Endpoints

### Solve Math Problem

<CodeGroup>
```bash POST /api/math-solver/solve
curl -X POST https://api.inspir.uk/api/math-solver/solve \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "problemText": "Solve for x: 2x + 5 = 15",
    "showSteps": true
  }'
```
</CodeGroup>

Solve a math problem with AI-powered step-by-step solution.

**Authentication:** Optional (guests get solution without saving)

#### Request Body

<ParamField path="problemText" type="string" required>
  The math problem to solve
</ParamField>

<ParamField path="showSteps" type="boolean" default={true}>
  Whether to include detailed step-by-step explanation
</ParamField>

#### Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="solution" type="object">
  Solution details
  
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Solution ID (null for guest users)
    </ResponseField>
    
    <ResponseField name="problemText" type="string">
      The original problem
    </ResponseField>
    
    <ResponseField name="problemType" type="string">
      Problem classification: algebra, calculus, geometry, trigonometry, statistics, or other
    </ResponseField>
    
    <ResponseField name="difficultyLevel" type="string">
      Difficulty: easy, medium, or hard
    </ResponseField>
    
    <ResponseField name="steps" type="array">
      Step-by-step solution
      
      <Expandable title="step object">
        <ResponseField name="stepNumber" type="number">
          Sequential step number
        </ResponseField>
        
        <ResponseField name="explanation" type="string">
          Explanation of the step
        </ResponseField>
        
        <ResponseField name="equation" type="string">
          Mathematical equation for this step
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="finalAnswer" type="string">
      The final answer to the problem
    </ResponseField>
    
    <ResponseField name="verification" type="string">
      Verification that the answer is correct
    </ResponseField>
  </Expandable>
</ResponseField>

### Get Solution History

<CodeGroup>
```bash GET /api/math-solver/history
curl -X GET https://api.inspir.uk/api/math-solver/history \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

Get user's math solution history (authenticated users only).

**Authentication:** Required

#### Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="solutions" type="array">
  Array of past solutions (up to 50 most recent)
  
  <Expandable title="solution object">
    <ResponseField name="id" type="string">
      Solution ID
    </ResponseField>
    
    <ResponseField name="problem_text" type="string">
      The problem that was solved
    </ResponseField>
    
    <ResponseField name="final_answer" type="string">
      The solution answer
    </ResponseField>
    
    <ResponseField name="problem_type" type="string">
      Problem classification
    </ResponseField>
    
    <ResponseField name="difficulty_level" type="string">
      Difficulty level
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

### Generate Practice Problems

<CodeGroup>
```bash POST /api/math-solver/solution/:id/practice
curl -X POST https://api.inspir.uk/api/math-solver/solution/123/practice \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "count": 3
  }'
```
</CodeGroup>

Generate similar practice problems based on a previously solved problem.

**Authentication:** Required

#### Path Parameters

<ParamField path="id" type="string" required>
  ID of the original solution to base practice problems on
</ParamField>

#### Request Body

<ParamField path="count" type="number" default={3}>
  Number of practice problems to generate (1-10)
</ParamField>

#### Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="problems" type="array">
  Generated practice problems with solutions
  
  <Expandable title="problem object">
    <ResponseField name="problem" type="string">
      The practice problem text
    </ResponseField>
    
    <ResponseField name="steps" type="array">
      Solution steps (same structure as solve endpoint)
    </ResponseField>
    
    <ResponseField name="answer" type="string">
      Final answer
    </ResponseField>
  </Expandable>
</ResponseField>

## Error Responses

<ResponseField name="error" type="string">
  Error message
</ResponseField>

Common error codes:
- `400` - Missing or invalid problem text
- `401` - Authentication required
- `404` - Solution not found
- `500` - AI service error or parsing failure

## Example Usage

```javascript
// Solve a math problem
const response = await fetch('https://api.inspir.uk/api/math-solver/solve', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    problemText: 'Find the derivative of f(x) = 3x^2 + 2x - 1',
    showSteps: true
  })
});

const data = await response.json();
console.log(data.solution.finalAnswer); // "f'(x) = 6x + 2"
console.log(data.solution.steps); // Step-by-step solution
```

## Supported Problem Types

- **Algebra** - Equations, systems, polynomials, factoring
- **Calculus** - Derivatives, integrals, limits
- **Geometry** - Area, perimeter, volume, angles
- **Trigonometry** - Identities, equations, graphing
- **Statistics** - Mean, median, standard deviation, probability
- **Other** - Number theory, combinatorics, logic

<Info>
The Math Solver uses Claude Sonnet 4.5 for accurate mathematical reasoning and can handle problems ranging from basic arithmetic to advanced calculus.
</Info>
