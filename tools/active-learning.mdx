# Active Learning Tools

Active learning tools help you test your knowledge and reinforce what you've learned. Instead of just reading or highlighting, these tools force you to recall, apply, and demonstrate your understanding.

<Info>
  Research shows that **active recall** is 2-3x more effective than passive review for long-term retention.
</Info>

## Flashcard Creator

Generate flashcard decks from any content with AI-powered spaced repetition.

### Creating Flashcards

<Tabs>
  <Tab title="From Text">
    1. Paste your notes or textbook content
    2. Choose number of cards (10-50)
    3. AI generates front/back pairs
    4. Edit cards as needed
  </Tab>
  
  <Tab title="From File">
    1. Upload PDF, DOCX, or TXT
    2. AI extracts key concepts
    3. Creates question/answer pairs
    4. Review and save deck
  </Tab>
  
  <Tab title="Manual Creation">
    1. Click "Add Card"
    2. Type front (question/term)
    3. Type back (answer/definition)
    4. Add to your deck
  </Tab>
</Tabs>

### Study Modes

<CardGroup cols={3}>
  <Card title="Flip Mode" icon="rotate">
    Classic flashcard flipping - see question, flip to reveal answer
  </Card>
  
  <Card title="MCQ Mode" icon="list">
    Turn cards into multiple choice questions with distractors
  </Card>
  
  <Card title="Type Mode" icon="keyboard">
    Type your answer before revealing - tests active recall
  </Card>
</CardGroup>

### Spaced Repetition

Flashcards use a spaced repetition algorithm to show you cards at optimal intervals:

- **New cards**: Shown frequently
- **Easy cards**: Shown less often (every 3-7 days)
- **Hard cards**: Shown more often (daily)
- **Mastered cards**: Reviewed weekly

<Warning>
  Mark cards as "Easy", "Good", or "Hard" after each review to optimize your learning schedule.
</Warning>

## Practice Test Builder

Create full-length practice exams that simulate real test conditions.

### Features

- **Timed tests**: Set time limits per question or for entire test
- **Mixed question types**: MCQ, short answer, true/false
- **Detailed scoring**: See which topics need more review
- **Performance analytics**: Track improvement over multiple attempts

### Best Practices

<Steps>
  <Step title="Create Comprehensive Tests">
    Use 20-50 questions covering all major topics from your exam
  </Step>
  
  <Step title="Simulate Test Conditions">
    Set a timer and work in a quiet environment without notes
  </Step>
  
  <Step title="Review Thoroughly">
    Study the explanation for every wrong answer
  </Step>
  
  <Step title="Retake Strategically">
    Wait 2-3 days between attempts to test long-term retention
  </Step>
</Steps>

## Mind Map Creator

Visualize concepts and their relationships with interactive mind maps.

### Creating Mind Maps

**From a topic:**
```
Input: "Cell Biology"

AI generates:
- Central node: Cell Biology
- Main branches: Cell Structure, Cell Function, Cell Division
- Sub-branches: Organelles, Membrane, Nucleus, etc.
```

**From uploaded notes:**
- Upload your notes (PDF/DOCX)
- AI identifies key concepts and relationships
- Generates hierarchical mind map
- Drag nodes to customize layout

### Layout Options

<Tabs>
  <Tab title="Tree Layout">
    Hierarchical structure with the main topic at the top and subtopics branching downward
  </Tab>
  
  <Tab title="Radial Layout">
    Central topic in the middle with concepts radiating outward
  </Tab>
</Tabs>

### Interactive Features

- **Drag & drop**: Rearrange nodes
- **Add nodes**: Click any node and add children
- **Color coding**: Assign colors to different topic categories
- **Export**: Download as image (PNG) or save to library

## Tips for Effective Active Learning

<CardGroup cols={2}>
  <Card title="Use Multiple Tools" icon="layer-group">
    Combine quizzes, flashcards, and mind maps for the same topic to reinforce from different angles
  </Card>

  </Card>

  </Card>
